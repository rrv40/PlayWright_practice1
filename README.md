
# 🚀 Playwright Automation Playground (E2E Testing Project)

This project demonstrates a complete End-to-End (E2E) automation framework built using **Playwright with JavaScript**.

It covers a wide range of real-world UI automation scenarios commonly used in automation projects, making it a great hands-on practice for learning modern web testing.

---

## ✅ 🔧 Tech Stack

- Playwright
- JavaScript (Node.js)
- HTML Test Automation Practice Website

---

## ✅ 📌 Features Covered

### 🔹 Form Handling
- Input fields (Name, Email, Phone, Address)
- Radio buttons
- Checkboxes

### 🔹 Dropdown Handling
- Single select dropdown (Country)
- Multi-select dropdown (Colors)
- Validation of dropdown options

### 🔹 ✅ Date Picker Automation
- Dynamic date selection using a reusable function
- Handles both past and future dates intelligently

### 🔹 File Upload
- Upload file using `setInputFiles()`
- Validate upload success

### 🔹 Search Automation
- Wikipedia search functionality automation

### 🔹 Dynamic Elements Handling
- Toggle button (Start → Stop → Start)
- Handling changing text and attributes

### 🔹 Alert Handling
- Simple Alert (OK)
- Prompt Alert (Input + Accept)

### 🔹 Tab Handling
- Open new tab
- Perform actions
- Switch back to parent tab

### 🔹 Popup Window Handling
- Handle child window using `page.waitForEvent('popup')`
- Switch between windows

### 🔹 Drag and Drop
- Using Playwright `dragTo()` API
- Validation after drop
